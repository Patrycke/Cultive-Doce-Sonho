# 💍 Cultive Doce Sonho – Assessoria e Cerimonial

> Site institucional desenvolvido para a empresa **Cultive Doce Sonho**, especializada em organização de eventos e casamentos na região da Grande Florianópolis.

Este projeto foi construído com foco em **Performance**, **SEO Local** e **UX (User Experience)**, utilizando práticas modernas de desenvolvimento front-end.

---

## 🛠️ Tecnologias Utilizadas

Para este projeto, foquei em uma estrutura leve e responsiva:

* **HTML5 & CSS3**: Semântica estruturada e estilização personalizada com variáveis (CSS Variables).
* **Bootstrap 5**: Framework para agilizar o layout responsivo e componentes de interface.
* **JavaScript (Vanilla)**: 
    * **Intersection Observer API**: Implementação de animações de *Scroll Reveal* (fade-in) de alta performance.
    * **WhatsApp API Integration**: Automação do formulário de orçamento com tratamento de strings e redirecionamento dinâmico.
    * **Regex / Input Masking**: Validação e máscara de telefone em tempo real para melhor usabilidade.
* **Google Fonts & Bootstrap Icons**: Tipografia refinada (Montserrat e Playfair Display) e ícones vetoriais.

---

## 🚀 Diferenciais Técnicos

### 1. Performance & UX
- **Smooth Scroll**: Navegação fluida entre as seções da Landing Page.
- **Mobile First**: Design totalmente adaptável para dispositivos móveis, garantindo que noivas e noivos acessem as informações de qualquer lugar.
- **Lazy Animations**: Elementos que surgem suavemente conforme o scroll, reduzindo a carga cognitiva e aumentando o engajamento.

### 2. Conversão (Marketing Digital)
- **SEO Local**: Meta tags otimizadas para busca orgânica em Florianópolis e Região.
- **Orçamento Direto**: Fluxo de contato simplificado via WhatsApp, reduzindo a fricção no funil de vendas da cliente.
